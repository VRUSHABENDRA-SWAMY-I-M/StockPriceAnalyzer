# 📊 Stock Price Analyzer

A Python project that fetches and analyzes stock market data.

---

## 🚀 Features

- Fetch stock price data
- Calculate Moving Average (MA)
- Analyze stock trends
- Display results in terminal

---

## 🛠 Tech Stack

- Python
- Pandas
- Matplotlib
- yfinance

---

## 📂 Project Structure

StockPriceAnalyzer/
│
├── main.py
├── requirements.txt
├── README.md

---

## ▶ How To Run

1. Clone the repository:

git clone https://github.com/VRUSHABENDRA-SWAMY-I-M/StockPriceAnalyzer.git

2. Go into project folder:

cd StockPriceAnalyzer

3. Install dependencies:

pip install -r requirements.txt

4. Run the project:

python main.py
